# Geopolitical Defense Fund Dashboard

Streamlit dashboard for a simulated Canadian/TSX-focused defense, aerospace, and geopolitical-risk portfolio.

The application lives in [`geopolitical_defense_fund/`](geopolitical_defense_fund/).

## Run Locally

```bash
cd geopolitical_defense_fund
python3 -m pip install -r requirements.txt
python3 -m streamlit run app.py
```

This is an educational simulation only. It is not financial advice and does not place real trades.
